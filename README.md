# 📅 La Mia Giornata - Tabella della Giornata

Un'applicazione web elegante e funzionale per gestire la propria routine quotidiana, eventi speciali e obiettivi, con elementi di gamification.

## 🚀 Funzionalità principali

- **Agenda Dinamica**: Gestione della routine per ogni giorno della settimana.
- **Eventi Speciali & Ricorrenze**: Supporto per compleanni, visite mediche e appuntamenti ricorrenti (giornalieri, settimanali, mensili, annuali).
- **Gamification**: Guadagna XP completando task, sali di livello e mantieni la tua streak giornaliera! 🏆
- **Dashboard Intelligente**:
    - Widget **"In Evidenza"**: Visualizza i prossimi eventi importanti.
    - **Riflessione del Giorno**: Uno spazio per note e gratitudine.
    - **Statistiche**: Monitora il tuo progresso settimanale.
- **Focus Timer**: Pomodoro timer integrato per ogni attività.
- **Design Premium**: Interfaccia moderna in stile Glassmorphism con temi dinamici basati sull'ora del giorno.
- **PWA Ready**: Installabile su smartphone come un'app nativa.

## 🛠️ Tecnologie utilizzate

- HTML5 (Semantico)
- CSS3 (Variabili, Grid, Flexbox, Animazioni)
- JavaScript (Vanilla, LocalStorage per la persistenza dati)
- Boxicons per l'iconografia.

## 📱 Installazione Mobile

L'app è una **Progressive Web App (PWA)**. Per installarla sul tuo telefono:
1. Apri il link nel browser (es. Chrome o Safari).
2. Seleziona "Aggiungi a schermata Home" dal menu del browser.
3. L'app apparirà tra le tue applicazioni!

## 💾 Backup dei dati

Puoi esportare ed importare i tuoi dati in formato JSON direttamente dal menu di modifica (icona matita ✎).
